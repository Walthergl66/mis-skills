# Naming Conventions

- Skill folders use lowercase kebab-case: `react-native-architecture`.
- `SKILL.md` frontmatter `name` must match the folder name.
- Prefer task or domain names over vendor names unless the vendor is the actual domain.
- Keep names short enough to type in prompts.
- Use adapter names only under `adapters/<agent>`.
- Do not put spaces, underscores, uppercase letters, or version numbers in skill folder names.

Good names:

- `api-design`
- `accessibility-checker`
- `github-actions-pro`
- `emil-design-eng`

Avoid:

- `API Design`
- `skill-v2`
- `cursor-react-native-architecture`
- `myAwesomeSkill`
